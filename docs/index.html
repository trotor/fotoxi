Fotoxi

A local tool for browsing, curating, and organizing large photo collections. Runs on your machine — your photos stay with you.

Paikallinen työkalu suurten kuvakokoelmien selaamiseen, karsimiseen ja järjestämiseen. Toimii omalla koneella — kuvat pysyvät sinulla.

Features

Ominaisuudet

🔍

Search & Filter

Haku ja suodatus

Search photos by text, camera, date, folder, or location. Powered by SQLite FTS5 full-text search.

Hae kuvia tekstillä, kameralla, päivämäärällä, kansiolla tai sijainnilla. FTS5-pohjainen kokotekstihaku.

🤖

AI Analysis

AI-analyysi

Local Ollama-powered image analysis: descriptions, tags, colors, and scene type. No cloud services needed.

Paikallinen Ollama-pohjainen kuva-analyysi: kuvaukset, tagit, värit ja kohtaustyyppi. Ei pilvipalveluja.

🔁

Duplicate Detection

Duplikaattien tunnistus

Perceptual hash and burst detection automatically find similar and near-identical photos.

Perceptual hash ja burst-tunnistus löytävät samanlaiset ja lähes identtiset kuvat automaattisesti.

📍

Location Search

Sijaintihaku

Find photos taken at the same place. Click any photo and select "nearby" to see what was shot nearby.

Etsi kuvia jotka on otettu samassa paikassa. Klikkaa kuvaa ja valitse "nearby" nähdäksesi lähellä otetut kuvat.

⌨️

Keyboard Shortcuts

Pikanäppäimet

Fast photo review: keep, reject, and browse with keyboard shortcuts — no mouse needed.

Nopea kuvien läpikäynti: hyväksy, hylkää ja selaa nuolinäppäimillä ilman hiirtä.

☁️

Cloud Compatible

Pilviyhteensopivuus

Works with iCloud and OneDrive folders. Automatically frees local space after processing.

Toimii iCloud/OneDrive-kansioiden kanssa. Vapauttaa tilaa automaattisesti käsittelyn jälkeen.

Guide: Culling Bad Photos

Käyttöohje: Huonojen kuvien karsiminen

Fotoxi's primary use case is helping you find and remove bad photos from a large collection.

Fotoxin tärkein käyttötarkoitus on auttaa löytämään ja poistamaan epäonnistuneet kuvat suuresta kokoelmasta.

1

Index your photos

Indeksoi kuvat

Add a photo folder and start indexing from the Indexing page. Fotoxi reads EXIF data, generates thumbnails, and optionally runs AI analysis.

Lisää kuvakansio ja käynnistä indeksointi Indexing-sivulta. Fotoxi lukee EXIF-tiedot, luo thumbnailit ja ajaa halutessasi AI-analyysin.

2

Browse the grid

Selaa kuvia ruudukossa

The Search page shows photos in a thumbnail grid. Narrow results with filters:

Search-sivulla näet kuvat thumbnailruudukkona. Voit rajata näkymää suodattimilla:

  • Folder – pick a specific folder
  • Camera – filter by camera model
  • Date – date range filter
  • Status – show only pending / kept / rejected
  • AI – show only AI-analyzed photos
  • Kansio – valitse tietty kansio
  • Kamera – rajaa kameran mukaan
  • Päivämäärä – aikavälisuodatus
  • Status – näytä vain pending/kept/rejected
  • AI – näytä vain AI-analysoidut kuvat
3

Open & review

Avaa kuva ja arvioi

Click a photo to open it full-size. Use keyboard shortcuts for fast review:

Klikkaa kuvaa avataksesi sen isona. Käytä pikanäppäimiä nopeaan arviointiin:

  • Enter – Keep photo & go to nextHyväksy kuva ja siirry seuraavaan
  • Backspace – Reject photo & go to nextHylkää kuva ja siirry seuraavaan
  • ← → – Previous / next photoEdellinen / seuraava kuva
  • Esc – Close viewerSulje kuvanäkymä
4

Review duplicates

Käy duplikaatit läpi

The Duplicates page shows similar photos in groups. Pick the best from each group and reject the rest in one go.

Duplicates-sivulla Fotoxi näyttää samankaltaiset kuvat ryhmittäin. Valitse joka ryhmästä paras kuva ja hylkää loput yhdellä kertaa.

Guide: Photos from the Same Location

Käyttöohje: Samassa paikassa kuvatut kuvat

Fotoxi can search photos by GPS location, so you can see everything shot at the same place or on the same trip.

Fotoxi osaa etsiä kuvia GPS-sijainnin perusteella, joten voit katsoa mitä kuvia on otettu samasta paikasta tai samalla reissulla.

1

Open a geotagged photo

Avaa kuva jossa on GPS-tieto

Photos with location data show a 📍 icon. Click one to open it.

Kuvat joissa on sijaintitieto näyttävät 📍-ikonin. Klikkaa kuvaa avataksesi sen.

2

Click "📍 nearby"

Klikkaa "📍 nearby"

In the photo details you'll see a [📍 nearby] link. Click it to find all photos taken near the same location.

Kuvan tiedoissa näkyy [📍 nearby] -linkki. Klikkaamalla sitä Fotoxi hakee kaikki kuvat jotka on otettu lähellä samaa sijaintia.

3

Adjust the radius

Säädä hakusädettä

Above the results you can adjust the search radius (default 500 m). Increase it to see a wider area, or decrease it for a tighter selection.

Sijaintihaun yläpuolella näkyy säädettävä hakusäde (oletuksena 500 m). Voit kasvattaa sädettä nähdäksesi laajemman alueen kuvat tai pienentää sitä tarkempaan rajaukseen.

1

Find photos from the same moment

Etsi samaan aikaan otetut kuvat

AI Analysis: Performance & Compatibility

AI-analyysi: suorituskyky ja yhteensopivuus

💻

Performance on Apple Silicon

Suorituskyky Apple Silicon -koneilla

AI analysis uses a local Ollama vision model to generate descriptions, tags, colors, and scene types for each photo. On a MacBook with an M-series chip (M1/M2/M3/M4), expect roughly 15–40 seconds per image depending on the model and chip variant. A collection of 1 000 photos takes approximately 6–10 hours to fully analyze. The analysis runs in the background, so you can browse and curate photos while it works.

AI-analyysi käyttää paikallista Ollama-visiomallia kuvausten, tagien, värien ja kohtaustyypin tuottamiseen jokaiselle kuvalle. M-sarjan MacBookilla (M1/M2/M3/M4) yhden kuvan analysointi kestää tyypillisesti 15–40 sekuntia mallista ja piiriversiosta riippuen. 1 000 kuvan kokoelman täysi analysointi vie noin 6–10 tuntia. Analyysi pyörii taustalla, joten voit selata ja järjestellä kuvia samaan aikaan.

🍎

macOS only (for now)

Vain macOS (toistaiseksi)

The local AI analysis with Ollama has been tested only on macOS with Apple Silicon. It may work on Linux or Windows with Ollama installed, but this has not been verified.

Paikallinen AI-analyysi Ollaman kanssa on testattu ainoastaan macOS-ympäristössä Apple Silicon -koneilla. Se saattaa toimia Linuxilla tai Windowsilla jos Ollama on asennettu, mutta tätä ei ole varmistettu.

🌐

Coming soon: External AI API

Tulossa: ulkoinen AI-rajapinta

Support for external AI APIs (such as OpenAI, Anthropic, or Google) is planned. This will enable faster analysis without local GPU requirements, and make AI features accessible on any platform.

Tuki ulkoisille AI-rajapinnoille (kuten OpenAI, Anthropic tai Google) on suunnitteilla. Tämä mahdollistaa nopeamman analysoinnin ilman paikallista GPU-vaatimusta ja tuo AI-ominaisuudet saataville kaikille alustoille.

⚡

Tip: Enable Ollama parallel processing

Vinkki: Ollaman rinnakkaiskäsittely

To speed up AI analysis, set OLLAMA_NUM_PARALLEL=2 (or higher on powerful machines) and increase the concurrency slider in Fotoxi Settings. On macOS:

AI-analyysin nopeuttamiseksi aseta OLLAMA_NUM_PARALLEL=2 (tai enemmän tehokkailla koneilla) ja nosta rinnakkaisuusliukusäädintä Fotoxin Asetuksissa. macOS:lla:

launchctl setenv OLLAMA_NUM_PARALLEL 2
# Restart Ollama (close menu bar icon and reopen, or:)
pkill ollama && open -a Ollama

Quick Start

Käynnistys

git clone https://github.com/trotor/fotoxi.git
cd fotoxi
python3 -m venv .venv && source .venv/bin/activate
pip install -e ".[dev]"
cd frontend && npm install && npm run build && cd ..
python fotoxi.py serve

The app starts at http://localhost:8000

Sovellus käynnistyy osoitteeseen http://localhost:8000
